**We are pleased** to share that the Flax Bourton Hub website has now been refreshed to make it clearer, easier to use and easier to keep up to date.

The new site brings together practical hire information, documents, activities, news and frequently asked questions in a simpler format so that residents, hirers and community groups can find what they need more quickly. It also gives trustees a much easier way to update routine content, helping the site stay current without relying on developer support for every change.

Importantly, the booking process itself is still based on the existing HallBookingOnline system, so there is no major change for anyone making a reservation. The website now helps people check information and availability more easily before they move through to the live booking system.

This is an important first step in improving how the Hub communicates with the village and supports community use. We hope the new site makes the Hub more welcoming, more useful and easier for everyone to navigate.
