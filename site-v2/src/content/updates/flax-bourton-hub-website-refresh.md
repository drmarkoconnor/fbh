**Welcome to the updated Flax Bourton Hub Website,** which is now simpler to use, easier to navigate, and fully update with the latest information. 

This new site brings together practical hire information, documents, activities, news and frequently asked questions in a simpler format so that residents, hirers and community groups can find what they need more quickly. It also gives Trustees a much easier way to update routine content, helping the site stay current without relying on developer support for every change.

Importantly, the booking process itself is still based on the existing online hall booking system, so there is no major change for anyone making a reservation. The website now helps people check information and availability more easily before they move through to the live booking system.

This is an important first step in improving how the Hub communicates with the village, and supports community use. 

We hope the new site makes the Hub more welcoming, more useful and easier for everyone to navigate.
